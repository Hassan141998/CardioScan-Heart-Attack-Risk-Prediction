# CardioScan ML Models

This directory holds trained model files generated by `train/train_model.py`.

Files (auto-generated, gitignored):
- `cardio_xgb.pkl`   — XGBoost classifier
- `cardio_rf.pkl`    — Random Forest classifier  
- `cardio_knn.pkl`   — KNN pipeline (with StandardScaler)

## To generate:
```bash
cd ../train
python train_model.py
```

The backend will automatically use heuristic fallback if .pkl files are missing.
